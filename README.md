# kddl
An archive of Caravel Games' Flash DROD, playable in modern browsers.

[Ruffle](https://ruffle.rs/) is used to make the game playable again.

The SWFs themselves are re-hosted by permission from Caravel Games, as they were offered for download (with this express purpose) [on their site](https://caravelgames.com/Articles/Games_2/FlashDROD/KDDL.html).
